# AI智能向导集成说明

## 📋 概述

本项目已集成智增增AI服务，提供智能向导推荐和问答功能。

## 🚀 功能特性

- **智能对话**: 基于智增增GPT模型的智能问答
- **向导推荐**: 根据用户需求智能推荐合适的向导
- **快速操作**: 预设常用问题，一键提问
- **降级处理**: API不可用时的友好降级方案
- **对话历史**: 保持上下文连贯性

## ⚙️ 配置说明

### 1. 环境变量配置

复制 `.env.example` 为 `.env.local`：

```bash
cp .env.example .env.local
```

编辑 `.env.local` 文件：

```env
# 智增增API密钥 - 替换为您的实际API密钥
VUE_APP_ZHIZENGZENG_API_KEY=sk-your-actual-api-key-here
```

### 2. API密钥获取

1. 访问 [智增增官网](https://zhizengzeng.com)
2. 注册账号并登录
3. 进入控制台获取API密钥
4. 将密钥配置到环境变量中

## 🔒 安全考虑

1. **API密钥保护**: 使用环境变量，不提交到代码库
